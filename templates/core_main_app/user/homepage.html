The MetaGenome Project is hosting an instance of the NIST Data Curation System, this system allows for the curation of Metamaterials Data in a repository using predefined templates.

This is being developed at the National Institute of Standards and Technology and is made available to solicit comments from the Material Science community. Please do not enter any proprietary data into this system.

You can find the MetaGenome Project GitHub Organisation here: Meta-Genome-Org.
